Supabase 연동 설정

부부 공유를 위해 Supabase 프로젝트 정보를 입력해 주세요.
한 번만 설정하면 됩니다!

STEP 1
supabase.com 에서 무료 계정 만들기 → 새 프로젝트 생성
STEP 2
Project Settings → API 에서 URL과 anon key 복사
STEP 3
SQL Editor에서 아래 버튼으로 테이블 생성 SQL 보기
Supabase SQL 보기
-- 거래 테이블 CREATE TABLE transactions ( id uuid DEFAULT gen_random_uuid() PRIMARY KEY, created_at timestamptz DEFAULT now(), date date NOT NULL, type text NOT NULL, category text NOT NULL, name text NOT NULL, amount integer NOT NULL, memo text, photo_url text, is_recurring boolean DEFAULT false, user_name text ); ALTER TABLE transactions ENABLE ROW LEVEL SECURITY; CREATE POLICY "Allow all" ON transactions FOR ALL USING (true); -- 예산 테이블 CREATE TABLE budgets ( id uuid DEFAULT gen_random_uuid() PRIMARY KEY, month text NOT NULL, category text NOT NULL, amount integer NOT NULL, UNIQUE(month, category) ); ALTER TABLE budgets ENABLE ROW LEVEL SECURITY; CREATE POLICY "Allow all" ON budgets FOR ALL USING (true);
